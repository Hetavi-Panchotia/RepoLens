
# 🚀 RepoLens – AI Codebase Explainer for Developers

## 🏆 Hackathon Project Submission

**RepoLens** is an AI-powered tool that helps developers quickly understand any GitHub repository by automatically generating summaries, documentation, folder explanations, and architecture insights.

Instead of spending hours reading unfamiliar code, RepoLens explains the project in seconds using Artificial Intelligence.

# 📌 Problem Statement

Developers often struggle to understand unfamiliar repositories when:

* Joining new teams
* Contributing to open-source projects
* Reviewing large codebases
* Working on legacy systems

Common challenges:

* Lack of proper documentation
* Complex folder structures
* Unknown architecture
* Time-consuming onboarding

This slows productivity and increases development time.

---

# 💡 Our Solution

**RepoLens** analyzes a GitHub repository using AI and automatically generates:

* Project Summary
* Folder & File Explanations
* Documentation
* Architecture Insights
* AI Chat with Codebase

This allows developers to understand any repository quickly and efficiently.

# 🧠 How It Works

1. User enters a GitHub repository URL
2. RepoLens fetches repository files using GitHub API
3. Backend sends code to AI model
4. AI analyzes the codebase
5. RepoLens generates insights
6. Results appear on dashboard
7. User can ask questions via AI chat
